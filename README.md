# Hotel Haven – AI Booking Cancellation Risk Dashboard

An interactive machine learning web application that predicts the probability of a hotel booking being cancelled.

This project demonstrates an end-to-end data science workflow including **data preprocessing, machine learning modelling and deployment of an interactive prediction dashboard using Streamlit**.

Users can enter booking information and receive a **real-time cancellation risk prediction** generated by a Random Forest model trained on historical hotel booking data.

## Project Overview

Booking cancellations are a major operational challenge within the hospitality industry. High cancellation rates can negatively impact revenue management, occupancy forecasting, and operational planning.

This project builds a predictive model that estimates the **probability that a booking will be cancelled** based on historical reservation data.

The application allows users to input booking details such as:

* Lead time before arrival
* Number of adults and children
* Room type
* Meal plan selection
* Market segment
* Number of nights booked
* Special requests
* Reservation timing

Based on these inputs, the model calculates a **cancellation risk probability**, which is displayed within the dashboard for operational decision-making.

---

## Machine Learning Approach

A **Random Forest Classifier** was used to train the cancellation prediction model.

Random Forest was selected because it:

* Handles non-linear relationships effectively
* Works well with tabular datasets
* Is robust to noise and overfitting
* Provides strong performance with minimal feature scaling

The model predicts the probability of cancellation using the `predict_proba()` method from scikit-learn.
